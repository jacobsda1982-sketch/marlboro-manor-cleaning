# Design System

## Brand character

Calm, precise, welcoming, local, refined, and trustworthy. Premium does not mean ornate.

## Color tokens

- Navy: `#1F365F`
- Gold: `#C7A247`
- White: `#FFFFFF`
- Cream: `#F8F6F2`
- Light gray: `#F5F6F7`
- Charcoal: `#3A3A3A`

Validate all text/background combinations against WCAG AA. Gold should generally be an accent rather than body text on white.

## Typography

- Display/headings: Cormorant Garamond or a performance-safe equivalent
- Body/UI: Montserrat or a performance-safe equivalent
- Use self-hosted or privacy-conscious loading where licensing permits.
- Maintain readable line lengths and fluid type scales.

## Layout

- Mobile-first
- Generous whitespace
- Clear content hierarchy
- Minimum 44×44 px interactive targets
- Consistent container widths and spacing tokens
- Sticky mobile action bar only if it does not obscure content

## Imagery

Preferred order:

1. Real founder/team/equipment/property-detail photography
2. Licensed architectural/interior imagery with no implied false proof
3. Neutral textures or design-led compositions

Never depict fictional branded employees, vehicles, reviews, or before/after work as authentic operations.

## Motion

Subtle and optional. Respect `prefers-reduced-motion`. Avoid scroll hijacking, long entrance delays, excessive parallax, or animation libraries without measurable benefit.
