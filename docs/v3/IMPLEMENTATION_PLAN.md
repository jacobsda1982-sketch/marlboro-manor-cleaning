# Implementation Plan

## Milestone 1 — Audit and foundation

- Audit V2 repository and deployment
- Inventory reusable brand/code assets
- Remove misleading operational imagery/content
- Establish typed business config, content model, tokens, and route framework

Exit: production build renders shell, navigation, footer, and safe placeholders.

## Milestone 2 — Core conversion pages

- Homepage
- Services overview and four service pages
- Pricing and quote-expectations page
- About, contact, FAQ, quote handoff

Exit: primary customer journey works on desktop/mobile without fabricated content.

## Milestone 3 — Local SEO and policies

- Initial high-value service-area pages
- Metadata, canonical, sitemap, robots, structured data
- Privacy, terms, cancellation, satisfaction, accessibility pages

Exit: all public routes are complete, unique, indexable, and internally linked.

## Milestone 4 — Integration and evidence

- Configure public Marble Quote Portal URL
- Add verified phone/profile/social links and factual claims
- Add authentic/approved imagery
- Add conversion analytics only if authorized

Exit: no placeholders or unverified claims remain in production mode.

## Milestone 5 — Hardening and release

- Automated tests/build/link/a11y/performance checks
- Manual device/browser testing
- Cloudflare preview deployment and owner review
- Release manifest, checksums, verification evidence
- Production deploy and post-launch checks only after owner approval
